Start of Blog

Posted on 2022-04-03 Edited on 2022-06-02 In blogging , start Disqus: Symbols count in article: 549 Reading time ≈ 1 mins.

¶Inspiration

很多时候感觉缘分就是如此奇妙。最近申请季结束开始研究Github（是有点晚了哈), 因为觉得命令行操作很酷就下载了git(谁不想了解这个世界上最先进的分布式管理系统呢), 虽然最终可能Github desktop的用户体验更好一点 当然日常生活中两个都会被用到;

又是机缘巧合在逛 appstore 的时候看到了mimo。当时只知道这是个教编程的软件，下载后一度被会员制/membership劝退，但最终发现只要使用的仔细认真，答题尽量不错，其实还是可以用的​（✈️ 起飞~）至此便开始了对于Web Development的学习: 从 html 到 css 到 javaScript. 在准备配置自己电脑vscode上的js编程环境的时候，下载的jsnode 搞了近一个小时 东搞搞西搞搞。

晚上睡觉前刷了刷b站，看到 up主 codesheep 上传的一篇视频 “如何0基础创建自己的博客“ 立马就感兴趣住了！说hexo配置的两个前置: git 和 jsnode 的时候 才意识到自己无意中已经给自己铺好了路。

搜集好教程后 准备第二天直接做起！

写到这里 我的博客基本配置已经ok啦 以及这就是我的第一篇文章。

希望以后的我看到这里能回想起当时那个 求知、热情、上进的自己 😏
