¶👉改变字体颜色: Text Colour

¶表示方法<value>:

  1. 十六进制值 如 #f1c40f
  2. RGB值 如 RGB(255,254,253)
  3. RGBA值 如 RGBA(255,0,0,0.5) 其中最后一个参数为alpha参数 代表不透明度 范围在0-1之间
  4. 颜色的名称 如 blue, lightpink

¶添加方法

  • [x] 在任意可以表达文字的html标签 (如<h1> ,<p>) 的开始标签内添加 style="corlor:<value>" 。

¶👉脚注: Footnote[1]

脚注可以在你想添加reference[2], 想解释一个词或者一句话但又不像占太多地方的时候被使用。

¶添加方法:

  • [x] 在正文中插入 [^1] , 当然这个1可以是任何数字 也可以是词语，比如[^脚注]
  • [x] 在文章的其他地方插入对应解释[^1]:解释，一般是在文末
    • [x] 在列表，表格，以及引用>this is a blockquote中无法添加
